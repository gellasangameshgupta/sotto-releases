# Sotto Releases

This repository hosts the update feed and releases for [Sotto](https://github.com/gellasangameshgupta/sotto) - a macOS voice-to-text dictation app.

## Download

Download the latest version from the [Releases](https://github.com/gellasangameshgupta/sotto-releases/releases) page.

## Update Feed

The app checks for updates via Sparkle using `appcast.xml` in this repository.
